# Product V2 - Vibecode Right-Click .sh Actions

> 셀링 전용 문서. 개발 README가 아니라 마켓플레이스 설명, 썸네일 기획, 소개 이미지 제작에 쓰는 메시지다.

## 한 줄 판매 문구

`.sh` 스크립트를 우클릭하거나 에디터 상단 플레이 버튼으로 바로 터미널에서 실행한다.

## 어떤 목적으로 쓰나

스크립트 위치로 이동하고 `bash ./script.sh`를 타이핑하는 반복을 없앤다. VSCode에서 셸 스크립트를 열거나 Explorer에서 선택한 상태 그대로 통합 터미널 실행까지 이어주는 용도다.

## 누구에게 팔아야 하나

- 레포 안에 빌드, 배포, 설치용 `.sh`가 많은 개발자
- 스크립트를 자주 테스트하는 자동화 담당자
- 터미널 경로 이동 실수를 줄이고 싶은 사람

## 사용법 시나리오

1. Explorer에서 `.sh` 파일을 우클릭한다.
2. Run Shell Script 액션을 선택한다.
3. 통합 터미널이 스크립트 폴더에서 열린다.
4. `bash ./script.sh`가 실행된다.
5. 열린 `.sh` 에디터에서는 상단 플레이 버튼으로 같은 동작을 실행한다.

## 썸네일 제작 지시

- 실제 그릴 장면: VSCode에서 `deploy.sh` 파일이 열려 있고, 에디터 상단 플레이 버튼을 누르자 아래 통합 터미널에서 스크립트가 실행되는 장면.
- 반드시 보여야 할 목적: "셸 스크립트를 클릭 한 번으로 실행한다"가 보여야 한다.
- 시각 요소: `.sh` 파일, 플레이 버튼, 우클릭 메뉴, 터미널 출력, 현재 작업 디렉토리.
- 피할 것: 터미널 아이콘 하나만 크게 그리기, 실제 VSCode 실행 흐름이 없는 이미지.
