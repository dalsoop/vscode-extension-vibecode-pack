# Product V2 - Vibecode Extension Menu List

> 셀링 전용 문서. 개발 README가 아니라 마켓플레이스 설명, 썸네일 기획, 소개 이미지 제작에 쓰는 메시지다.

## 한 줄 판매 문구

설치된 모든 `vibecode-*` 확장과 명령을 한 사이드바에 모아 클릭 한 번으로 실행한다.

## 어떤 목적으로 쓰나

Vibecode 확장이 늘어날수록 명령 팔레트에서 이름을 기억해 실행하기가 어려워진다. 이 확장은 설치된 Vibecode 확장을 자동 스캔하고, 각 command를 카드와 검색 가능한 카탈로그로 보여주는 런처 역할을 한다.

## 누구에게 팔아야 하나

- Vibecode 확장을 여러 개 설치한 사용자
- 명령 팔레트보다 시각적인 런처를 선호하는 개발자
- 자체 `vibecode-*` 확장을 배포해 사용자에게 한곳에서 실행시키고 싶은 팀

## 사용법 시나리오

1. Activity Bar에서 Vibecode V 아이콘을 누른다.
2. 사이드바에서 설치된 Vibecode 확장 목록을 본다.
3. 필터에 키워드를 입력해 원하는 명령을 찾는다.
4. 카드 클릭으로 즉시 VSCode command를 실행한다.
5. 새 확장을 설치하면 자동 새로고침으로 목록에 반영된다.

## 썸네일 제작 지시

- 실제 그릴 장면: VSCode Activity Bar의 V 아이콘을 누르자 사이드바에 여러 Vibecode 확장 카드와 command 버튼들이 펼쳐진 장면.
- 반드시 보여야 할 목적: "Vibecode 명령들을 한곳에서 실행한다"가 보여야 한다.
- 시각 요소: 확장별 카드 섹션, 검색 필터, command 버튼, 클릭 커서, 자동 갱신 표시.
- 피할 것: V 로고만 크게 그리기, 앱 아이콘 나열만 하기, 실제 명령 실행 UI 없는 이미지.
