# Changelog

## 0.1.0

- Initial release.
- Custom editor for PNG / JPG / JPEG / WEBP / GIF / BMP / AVIF / HEIC / HEIF / SVG / ICO.
- Bottom metadata panel: file info card, camera/lens/exposure summary card, GPS coordinates with Google Maps link, full EXIF raw dump (expandable).
- Zoom (fit / 100% / +/-), background toggle (checker / dark / light).
