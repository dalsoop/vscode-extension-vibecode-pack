# Vibecode Right-Click — Open / Reveal Actions

> **파일·폴더 우클릭에 자주 쓰는 5가지 액션을 한 번에.**
> 브라우저로 열기 / Finder reveal / 절대경로 복사 / 새창 / 터미널.

---

## 누구를 위한 거?

- 매번 `code path/to/file` 따로 치기 귀찮은 멀티프로젝트 작업자
- HTML / SVG / PDF 빨리 외부 도구로 열어보고 싶은 디자이너 / QA
- "파일을 Finder 에서 보고 싶다" 가 일상인 macOS 사용자
- 폴더 우클릭으로 새 VSCode 창을 자주 여는 모노레포 사용자

## 풀어주는 문제

| 평소 흔한 상황 | 이 확장으로 |
|---|---|
| `.html` 파일을 기본 브라우저로 열고 싶음 → 텍스트로만 열림 | 우클릭 → **기본 브라우저로 열기** (HTML / SVG / PDF) |
| 이 파일을 Finder 에서 보고 싶음 → `⌘⇧.` 트릭 외워야 함 | 우클릭 → **Finder / Explorer 에서 열기** |
| 절대경로를 다른 도구에 붙여넣어야 함 | 우클릭 → **절대 경로 복사** |
| 모노레포에서 서브폴더만 새 창으로 열고 싶음 | 폴더 우클릭 → **새 VSCode 창에서 폴더 열기** |
| 통합 터미널을 이 폴더 경로에서 열고 싶음 | 폴더 우클릭 → **여기서 터미널 열기** |

## 핵심 셀링 포인트

1. **5개 액션 한 그룹** — 컨텍스트 메뉴 하단의 `6_rca@NN` 그룹에 묶여서 표시. 위치 일관.
2. **Explorer + Editor Tab 양쪽 노출** — 같은 액션을 파일 트리에서도, 탭에서도 사용 가능.
3. **`when` clause 정확** — HTML/SVG/PDF 만 "브라우저로 열기" 등 적용 대상 자동 필터.
4. **manifest-driven 모듈 구조** — 액션 1개 = 폴더 1개 (`src/apps/<name>/`). 새 액션 추가는 폴더만 만들면 됨.
5. **i18n 1파일 = 1언어** — `i18n/<locale>.json` 추가로 새 언어 즉시 지원. 영문 기본값은 `nls-defaults.json`.

## 5개 액션 한눈에

```
┌─ 파일 우클릭 ───────────────────────┐
│  ... (기본 메뉴들)                  │
│  ─────────────────────────────     │
│  ⊕ 기본 브라우저로 열기            │ ← .html / .svg / .pdf
│  ▾ Finder 에서 열기                │
│  ⎘ 절대 경로 복사                  │
│  ─────────────────────────────     │
└──────────────────────────────────┘

┌─ 폴더 우클릭 ───────────────────────┐
│  ... (기본 메뉴들)                  │
│  ─────────────────────────────     │
│  ▾ Finder 에서 열기                │
│  ▢ 새 VSCode 창에서 폴더 열기      │
│  ⌨ 여기서 터미널 열기              │
│  ─────────────────────────────     │
└──────────────────────────────────┘
```

## 썸네일 아이디어

- **메인 컷**: 파일 위에 우클릭 메뉴가 펼쳐진 모습 — 우리 그룹의 5개 행을 하이라이트
- **각 행 옆 아이콘**: ⊕ ▾ ⎘ ▢ ⌨ (브라우저 / Finder / 클립보드 / 새 창 / 터미널)
- **컬러 키**: VSCode 다크 배경 + 우리 그룹만 살짝 elevated 강조 + 시안 하이라이트 라인
- **부제**: "5 right-click actions · One group"
- **사이드**: 메뉴에서 시작한 화살표 5개가 각각 Finder / Browser / Terminal 등의 아이콘으로 흩어지는 구도
