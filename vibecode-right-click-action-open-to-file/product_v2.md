# Product V2 - Vibecode Right-Click Actions

> 셀링 전용 문서. 개발 README가 아니라 마켓플레이스 설명, 썸네일 기획, 소개 이미지 제작에 쓰는 메시지다.

## 한 줄 판매 문구

파일과 폴더에서 자주 하는 열기, 경로 복사, Finder 보기, 터미널 열기를 우클릭 메뉴로 바로 실행한다.

## 어떤 목적으로 쓰나

개발 중 반복되는 OS 이동과 경로 복사를 VSCode Explorer 안에서 끝낸다. 브라우저로 열기, 새 창에서 폴더 열기, OS 파일 매니저에서 보기, 절대 경로 복사, 터미널 열기를 한 묶음으로 제공한다.

## 누구에게 팔아야 하나

- VSCode Explorer에서 파일과 폴더를 많이 다루는 개발자
- HTML/PDF/SVG를 바로 브라우저에서 열어 확인하는 사람
- 터미널, Finder, 경로 복사를 반복하는 작업자

## 사용법 시나리오

1. Explorer 또는 열린 에디터 탭에서 파일/폴더를 우클릭한다.
2. 필요한 Vibecode 액션을 선택한다.
3. 브라우저, Finder/Explorer, 터미널, 새 VSCode 창 중 원하는 흐름이 바로 실행된다.
4. 경로가 필요하면 Copy Absolute Path로 클립보드에 복사한다.

## 썸네일 제작 지시

- 실제 그릴 장면: VSCode Explorer에서 파일을 우클릭한 컨텍스트 메뉴에 "Open in Browser", "Reveal in Finder", "Copy Path", "Open Terminal Here"가 보이는 장면.
- 반드시 보여야 할 목적: "파일 작업 액션을 우클릭으로 빠르게 실행한다"가 보여야 한다.
- 시각 요소: 우클릭 메뉴, 선택된 파일, 브라우저/터미널/Finder가 작은 결과 창으로 열리는 모습.
- 피할 것: 마우스 아이콘 하나만 그리기, 액션 목록이 보이지 않는 추상 이미지.
