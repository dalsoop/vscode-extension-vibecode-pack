# Product V2 - Vibecode Skills Viewer

> 셀링 전용 문서. 개발 README가 아니라 마켓플레이스 설명, 썸네일 기획, 소개 이미지 제작에 쓰는 메시지다.

## 한 줄 판매 문구

Claude, Codex, Copilot, Cursor, Windsurf, Cline 스킬을 한 사이드바에서 검색하고 미리보고 instruction file에 동기화한다.

## 어떤 목적으로 쓰나

AI 코딩 도구의 스킬과 룰이 여러 폴더에 흩어지는 문제를 해결한다. 전역, 워크스페이스, 확장 번들 스킬을 한 곳에서 보고, 필요한 스킬을 instruction file에 관리 블록으로 넣어 프로젝트 컨텍스트를 맞추는 용도다.

## 누구에게 팔아야 하나

- 여러 AI 코딩 에이전트를 함께 쓰는 개발자
- 팀 스킬/룰을 표준화해 프로젝트에 배포하려는 리드
- 어떤 스킬이 어디에 설치됐는지 자주 잊는 사용자

## 사용법 시나리오

1. Activity Bar에서 Vibecode Skills를 연다.
2. Claude, Codex, Cursor 같은 tool chip으로 목록을 필터링한다.
3. 스킬을 클릭해 `SKILL.md` 프리뷰를 본다.
4. 즐겨찾기 또는 검색으로 자주 쓰는 스킬을 찾는다.
5. Sync to Instruction Files로 `CLAUDE.md`, `AGENTS.md`, `.cursorrules` 등에 관리 블록을 쓴다.

## 썸네일 제작 지시

- 실제 그릴 장면: VSCode 사이드바에 여러 도구 칩과 스킬 트리가 있고, 오른쪽에는 선택한 `SKILL.md` 미리보기와 instruction file sync 결과가 보이는 장면.
- 반드시 보여야 할 목적: "여러 AI 도구 스킬을 한곳에서 관리한다"가 보여야 한다.
- 시각 요소: Claude/Codex/Cursor/Windsurf 필터 칩, 스킬 목록, Markdown 프리뷰, sync 화살표, `AGENTS.md` 관리 블록.
- 피할 것: AI 로고 나열만 하기, 실제 스킬 목록이 없는 추상 이미지.
