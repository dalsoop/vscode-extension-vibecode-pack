# Changelog

## 0.1.0

- Initial release.
- Custom editor for `.pdf` — auto-opens on click.
- Page rendering via pdf.js, page navigation (prev/next/jump-to/total), zoom (fit-width / fit-page / 50–400% in steps).
- Per-page text extraction → copy to clipboard.
- Metadata sidebar: title, author, subject, keywords, creator, producer, created/modified dates, PDF version, page count, encrypted flag.
- Outline (TOC) panel with clickable jump-to-destination.
