# Vibecode .env Viewer (Import Only)

> **`.env` 파일을 paste-only 보안 뷰로 연다.**
> 키는 보이지만 **값은 어떤 경우에도 화면에 표시되지 않는다.**

---

## 누구를 위한 거?

- 코드 화면 공유 / 발표 / 페어 프로그래밍 중에 시크릿 노출 사고가 무서운 사람
- 어깨너머 (shoulder-surfing) 가 일상인 오픈 오피스 / 카페 작업자
- `.env` 를 평문 그대로 두긴 두되, **눈에는 안 띄게** 하고 싶은 1인 개발자
- 화면 녹화 / 스크린샷 / 캡처툴 활성화돼 있는 환경에서 시크릿 입력해야 하는 상황

> 디스크 자체를 암호화하고 싶으면 → [`vibecode-env-viewer-encryption-import-only`](../vibecode-env-viewer-encryption-import-only/) 참고. 위협 모델이 다르다.

## 풀어주는 문제

| 평소 흔한 상황 | 이 확장으로 |
|---|---|
| 텍스트 에디터로 `.env` 열면 값이 전부 평문 | **값은 화면에 절대 안 표시.** `··· (설정됨)` 또는 `(비어있음)` 만 |
| 값 길이만 봐도 패스워드 길이 노출 | **길이도 노출 안 함** (마스크는 고정 점 3개) |
| 키보드로 입력하다가 실수로 다른 값 덮어씀 | **키 입력 차단**. `⌘V` / `Ctrl+V` 만 허용 |
| paste 후 input 에 값이 남아있어서 화면에 잠깐 보임 | 붙여넣는 즉시 `input.value = ''` 로 지워짐 |
| 새 KEY 추가는 어떻게? | 비상 우회: `Open With… → Text Editor` 로 일반 편집 가능 |

## 핵심 셀링 포인트

1. **자동 인터셉트** — `.env` 더블클릭하면 자동으로 보안 뷰 (text editor 안 열림).
2. **paste-only 입력** — `keydown` / `beforeinput` 양쪽에서 키 입력 차단. ⌘V 만.
3. **화면 노출 0** — input 값이 DOM 에 남지 않음. 길이도 노출 안 함.
4. **webview CSP 잠금** — `default-src 'none'` + nonce. 외부 fetch 불가.
5. **비상 우회** — `Reopen Editor With… → Text Editor` 한 번에 일반 편집 모드.
6. **VSCode 표준 흐름** — `⌘S` / `Ctrl+S` 로 저장. 별도 학습 0.

## 사용 흐름 (시각)

```
Explorer
  ■ .env  ◄── 더블클릭

           ↓ (자동 인터셉트)

┌─────────────────────────────────────────────┐
│ ▦ .env (Import Only)                        │
├─────────────────────────────────────────────┤
│  KEY              VALUE                      │
│  ────────────    ──────────────────────      │
│  DATABASE_URL    [ ⌘V 로 붙여넣기 ]  ··· (설정됨)│
│  API_KEY         [ ⌘V 로 붙여넣기 ]  ··· (설정됨)│
│  STRIPE_SECRET   [ ⌘V 로 붙여넣기 ]  (비어있음)  │
│  DEBUG           [ ⌘V 로 붙여넣기 ]  ··· (설정됨)│
└─────────────────────────────────────────────┘
     ↑                          ↑
  키보드 입력 차단        값은 절대 화면에 안 표시
```

## 썸네일 아이디어

- **메인 컷**: 일반 텍스트 에디터에 `.env` 가 평문 펼쳐진 모습 (왼쪽, 흐릿하게) ↔ 이 확장의 보안 뷰 (오른쪽, 선명) — 좌측은 빨간 ✗, 우측은 그린 ✓
- **포인트 글자**: 우측 패널에 `DATABASE_URL  ⌘V  ••• (set)` 식 행이 3-4줄
- **컬러 키**: 다크 배경 + 그린 방패 ▦ + 마스크 점 `•••`
- **부제**: "Paste-only · Eyes-off"
- **사이드**: 키보드 아이콘 위에 ✗ 마크 (입력 차단), 클립보드 아이콘 옆에 ✓ (paste 만 허용)
