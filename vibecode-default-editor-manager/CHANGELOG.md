# Changelog

## 0.1.0

- Initial release.
- Sidebar webview view inside the `Vibecode` activity bar container.
- Lists every installed custom editor discovered via `contributes.customEditors` (viewType + display name + selectors + priority + source extension).
- Shows current `workbench.editorAssociations` as a pattern → editor table.
- Add new mapping (pattern + editor dropdown) and remove existing mapping.
- Scope picker on every write: Workspace or User settings.
